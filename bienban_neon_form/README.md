# Bảng Xác Nhận Sản Xuất - Neon + PDF

- Giao diện **Neon xanh dương** (nền tối, viền phát sáng).
- **Lưu dữ liệu** vào LocalStorage.
- **Xuất PDF** đầy đủ thông tin & hình ảnh (html2canvas + jsPDF).
- Up **thẳng** lên GitHub Pages (static).

## Deploy nhanh
1. Tạo repo GitHub (public), upload các file trong thư mục này.
2. **Settings → Pages**: chọn Branch `main`, folder `/root` → Save.
3. Truy cập: `https://<username>.github.io/<repo>/`.

## Sử dụng
- Điền form → (tuỳ chọn) vẽ chữ ký → **Lưu & Gửi** để lưu vào trình duyệt.
- Nhấn **In / Tải PDF** để xuất biên bản đầy đủ (ảnh + chữ ký).
